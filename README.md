## Description


## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Running migration script
```bash
$ ./migration.sh db/migration/name-of-migration
```

### give execution permission to the script
```bash
$ chmod +x ./migration.sh
```
